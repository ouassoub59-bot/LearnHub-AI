# LearnHub-AI 🚀

**Advanced AI Educational Platform for Global Students**

A comprehensive web-based learning platform powered by cutting-edge AI technology (Google Gemini API), designed to provide personalized education for students worldwide.

## Features ✨

- 🤖 **Advanced AI Tutor** - Gemini-powered assistant that:
  - Answers questions on lessons
  - Explains complex concepts
  - Generates AI videos for topics
  - Creates personalized presentations
  - Provides adaptive exercises
  - Professional calligraphy generation

- 👥 **Multi-language Support** - Global accessibility
- 🔐 **Admin Dashboard** - Special access for `ouassoub59@gmail.com`
- 📊 **Progress Tracking** - Monitor student performance
- 🎨 **Interactive UI** - Modern and responsive design
- 🎬 **AI Video Generation** - Auto-generated educational videos
- 📝 **Exercise Generation** - Adaptive problem sets

## Quick Start 🚀

### Prerequisites
- Node.js 18+
- MongoDB
- Docker & Docker Compose
- Google Gemini API Key
- Firebase Project

### Installation

```bash
# Clone repository
git clone https://github.com/ouassoub59-bot/LearnHub-AI.git
cd LearnHub-AI

# Setup environment
cp .env.example .env

# Install dependencies
npm install

# Start with Docker
docker-compose up

# Or start locally
npm run dev:backend  # Terminal 1
npm run dev:frontend # Terminal 2
